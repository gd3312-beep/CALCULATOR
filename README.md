# Modern Calculator

A sleek Java Swing calculator with a dark theme, rounded buttons, and hover effects. Supports basic arithmetic, percentage, sign toggle, history, and keyboard input.

## Features
- Operations: `+`, `-`, `*`, `/`, `%`
- Clear (`C`), backspace (`←`), decimal (`.`), sign toggle (`±`)
- Keyboard support: numbers, operators, Enter/`=`, Backspace, ESC
- Displays last operation in history
- Modern UI with rounded buttons and hover effect

## Run
Compile and run the Java file:

```bash
javac ModernCalculator.java
java ModernCalculator
